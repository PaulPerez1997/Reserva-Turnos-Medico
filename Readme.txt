Saludos! Agradezco su visita a este proyecto.

Este proyecto fue echo como practica de un curso de .NetCore, MVC, EntityFramework.
el objetivo de este proyecto es realizar una aplicacion web de reservas de turnos medicos, 
que cuenta con los CRUD para las entidades Medico, Paciente, Especialidades.

El sistema tiene un calendario de JavaScript ( https://fullcalendar.io ) este calendario utiliza Jquery y Ajax. 

El motor de base de datos que utilize fue SQL Express, 
fue construida la base de datos con EntityFramework y a medida que se requeria se utilizaba la herramienta Migrations.

La aplicacion posee un Login para el admin,
dicha password para ingresar esta compuesta con encriptacion SHA1 que proporciona SQL.

Una vez iniciado, el admin puede ver Turnos de los Medicos en el Calendario, Asignar turno o Eliminar el turno.
Para el Front se utilizo Materialize, que tambien provee una vista totalmente Responsive.
